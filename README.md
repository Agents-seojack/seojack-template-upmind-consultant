# Upmind Consultant — SEOJack Template Factory

A Next.js consulting website template inspired by the section structure and polished Webflow-style presentation of Upmind Temlis.

## Stack

- Next.js App Router
- TypeScript
- CSS custom properties
- Geist font package
- Vercel-ready configuration

## Local development

```bash
npm install
npm run lint
npm run typecheck
npm run build
npm run dev
```

## SEOJack CTA

Generic template ownership CTAs point to:

```text
https://seojack.net
```

Override locally with:

```bash
NEXT_PUBLIC_SEOJACK_CTA_URL=https://seojack.net
```
